BPI (BOAT PRO INDEX) クリックキット v7

■ やること（これだけ）
1) ZIPを右クリック →「すべて展開」
2) 展開したフォルダを開く
3) START.bat をダブルクリック

■ 何が起きる？
- まずデータ更新（data\status.json / data\summary.json / data\races_flat.json を作成）
- その後、ビューア（localhost）を開きます
- さらに（可能なら）毎日 02:10 に自動更新するタスクを登録します

■ 更新方法
- 手動更新：START.bat をもう一回ダブルクリック
- 自動更新：タスクが入っていれば毎日 02:10 に勝手に更新

■ よくある原因
- ZIPの中で直接実行すると失敗します → 必ず「すべて展開」してから
- OneDrive同期が重い時は data\*.json の作成が少し遅れます（少し待ってからページ更新）
